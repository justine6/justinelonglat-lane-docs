Notes

This release introduces the Platform documentation layer for the JLT ecosystem. The platform documentation defines the access-control model, protected resource boundaries, request lifecycle, and platform delivery flow. This documentation serves as the control-plane reference for how the JLT platform is designed, secured, and operated.
